### Para rodar os exemplos
No Ubuntu
```
	sudo apt-get install python-pip
```
No MacOS
```
	sudo easy_install pip
```
Com o pip instalado

```
	sudo pip install virtualenv
	mkdir ~/.virtualenvs
	cd ~/.virtualenvs
	virtualenv se2007
	source ~/.virtualenvs/se2007/bin/activate
```
